# Training & Problem Solving Website

موقع تعليمي تفاعلي يوفر منصة للتدريب وحل المشكلات البرمجية.

## 🌟 المميزات

- تصميم عصري وجذاب
- واجهة مستخدم سهلة الاستخدام
- تنظيم واضح للمحتوى التعليمي
- دعم كامل للغة العربية والإنجليزية
- تجربة مستخدم سلسة وتفاعلية

## 📁 هيكل المشروع

```
Trainin_Problem_Solving/
├── components/           # مكونات الصفحة الرئيسية
│   ├── header.html      # شريط التنقل العلوي
│   ├── hero.html        # قسم الترحيب الرئيسي
│   ├── main.html        # المحتوى الرئيسي
│   └── footer.html      # تذييل الصفحة
├── css/                 # ملفات التنسيق
│   ├── style.css        # التنسيقات العامة
│   ├── header.css       # تنسيقات شريط التنقل
│   ├── hero.css         # تنسيقات قسم الترحيب
│   ├── main.css         # تنسيقات المحتوى الرئيسي
│   └── footer.css       # تنسيقات التذييل
├── js/                  # ملفات JavaScript
│   ├── main.js          # الوظائف الرئيسية
│   └── scroll-top.js    # زر العودة للأعلى
├── pages/               # الصفحات الإضافية
│   ├── about.html       # صفحة "من نحن"
│   ├── contact.html     # صفحة الاتصال
│   ├── courses.html     # صفحة الدورات
│   └── resources.html   # صفحة المصادر
└── index.html           # الصفحة الرئيسية
```

## 🛠️ التقنيات المستخدمة

- **HTML5**: هيكل الصفحات
- **CSS3**: التنسيق والتصميم
  - Flexbox & Grid للتخطيط
  - CSS Variables للألوان والقيم المتكررة
  - Animations للحركات والتأثيرات
- **JavaScript**: التفاعلية والوظائف
- **Bootstrap 5.3.3**: إطار العمل الأساسي
- **Font Awesome 6.0.0**: الأيقونات

## 📝 خطوات التطوير

1. **إنشاء هيكل المشروع**
   - إنشاء المجلدات الأساسية
   - تنظيم الملفات حسب الوظيفة

2. **تطوير الصفحة الرئيسية**
   - إنشاء شريط التنقل العلوي
   - تصميم قسم الترحيب الرئيسي
   - إضافة أقسام المحتوى
   - تصميم تذييل الصفحة

3. **تطوير الصفحات الإضافية**
   - صفحة الدورات
   - صفحة المصادر
   - صفحة من نحن
   - صفحة الاتصال

4. **تحسين التصميم**
   - إضافة التأثيرات الحركية
   - تحسين الألوان والتباين
   - تطوير تجربة المستخدم

5. **التوثيق وإدارة الإصدارات**
   - إعداد ملف README
   - تهيئة نظام Git
