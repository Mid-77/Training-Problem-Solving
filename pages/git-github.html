دليل Git و GitHub

أساسيات Git

الأوامر الأساسية

الأمر الوصف مثال
git init إنشاء مستودع Git جديد git init
git add إضافة ملفات للمتابعة git add .
git commit إنشاء نقطة مرجعية git commit -m "وصف التغييرات"
git status عرض حالة المستودع git status
git log عرض سجل النقاط المرجعية git log --oneline

العمل مع GitHub

ربط المستودع المحلي مع GitHub

  1. إنشاء حساب على GitHub
  2. إنشاء مستودع جديد على GitHub
  3. ربط المستودع المحلي: git remote add origin https://github.com/username/repo.git
  4. رفع الكود لأول مرة: git push -u origin master

العمل مع الفروع (Branches)

الأمر الوصف
git branch عرض قائمة الفروع
git branch name إنشاء فرع جديد
git checkout name الانتقال إلى فرع
git merge name دمج فرع مع الفرع الحالي
